# EDD SSLCommerz Gateway

[![License](https://img.shields.io/badge/license-GPL--2.0--or--later-blue.svg)](https://www.gnu.org/licenses/gpl-2.0.html)
[![WordPress](https://img.shields.io/badge/wordpress-6.0%2B-blue)](https://wordpress.org/)
[![EDD](https://img.shields.io/badge/edd-3.0%2B-orange)](https://easydigitaldownloads.com/)

Integrate **SSLCommerz**, the first and largest payment gateway in Bangladesh, into your **Easy Digital Downloads** store. Accept payments via local cards (VISA, Mastercard, Amex), Mobile Banking (bKash, Rocket, Nagad), and Net Banking.

## 🚀 Features

- **Dual Checkout Modes**: 
  - **Hosted Redirect**: Redirect customers to the secure SSLCommerz payment page.
  - **Refund Processing**: Initiate refunds directly from the Easy Digital Downloads payment history.
- **Secure IPN**: Robust Handling of SSLCommerz Instant Payment Notifications with signature verification and data integrity checks.
- **Optimized Performance**: Efficient transaction mapping using industrial-strength meta queries.
- **Production Ready**: Built with security best practices, including TLS 1.2 enforcement and input sanitization.
Cryptographic signature verification and order verification via SSLCommerz API.
- **Refund Integration**: Process refunds directly from the EDD payment history screen.
- **Sandbox Support**: Easy testing with the official SSLCommerz Sandbox environment.
- **Developer Friendly**: Clean, modular code following WordPress and EDD coding standards.

## 🛠️ Installation

1. Download the latest release `.zip`.
2. Go to **Plugins > Add New** in your WordPress dashboard.
3. Upload the `.zip` file and click **Install Now**.
4. **Activate** the plugin.

## ⚙️ Configuration

1. Navigate to **Downloads > Settings > Payments > SSLCommerz**.
2. **Enable** the gateway.
3. Enter your **Store ID** and **Store Password** (provided by SSLCommerz).
4. Choose your preferred **Checkout Mode** (Hosted or Popup).
5. Toggle **Sandbox Mode** if you are testing.
6. Click **Save Changes**.
