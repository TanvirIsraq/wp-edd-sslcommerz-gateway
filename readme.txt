Add SSLCommerz payment gateway support to Easy Digital Downloads.

== Description ==

This plugin integrates SSLCommerz payment gateway with Easy Digital Downloads (EDD), allowing you to accept payments via cards, mobile banking, and internet banking in Bangladesh and other supported countries.

**Key Features:**
- Support for both Hosted Redirect and Popup (EasyCheckout) payment modes
- Secure payment processing with SSLCommerz API
- Full refund support from EDD admin
- Detailed transaction logging
- Sandbox mode for testing
- Supports multiple currencies (BDT, USD, EUR, GBP, SGD)
- Comprehensive admin interface for transaction details
- Debug logging for troubleshooting

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/edd-sslcommerz-gateway` directory, or install the plugin through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Downloads → Settings → Payment Gateways and configure your SSLCommerz credentials

== Frequently Asked Questions ==

= What payment methods are supported? =
SSLCommerz supports Visa, Mastercard, American Express, Mobile Banking (bKash, Nagad, Rocket), Internet Banking, and more.

= Which currencies are supported? =
BDT (Bangladeshi Taka), USD (US Dollar), EUR (Euro), GBP (British Pound), and SGD (Singapore Dollar).

= How do I get SSLCommerz credentials? =
Sign up for a merchant account at [SSLCommerz](https://www.sslcommerz.com/) to get your Store ID and Store Password.

= Can I test payments before going live? =
Yes, enable Sandbox Mode in the plugin settings and use SSLCommerz test credentials.
