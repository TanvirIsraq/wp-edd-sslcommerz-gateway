# .agent Documentation Index

This folder contains the **Agent-Assisted Development (AAD)** documentation for `wp-edd-sslcommerz-gateway`. It serves as a "living brain" for both AI agents and human developers who want to understand the deep architecture and implementation rationale.

## 🤖 Purpose
- **Architecture Source of Truth**: Detailed file-by-file behavior documentation.
- **Agent Handoffs**: Standardized context for AI collaborators.
- **Verification History**: Walkthroughs and regression checklists used during development.
- **Reference Alignment**: Keeping code aligned with SSLCommerz v4 and EDD API specs.

## 📂 Documentation Manifest
- `implementation-guide.md`: **Read this first.** Current architecture and file-by-file behavior.
- `walkthrough.md`: Step-by-step instructions for end-to-end testing and verification.
- `PLAN.md`: The roadmap used during the initial build (now in Maintenance phase).
- `tasks.md`: Historical task list and completed milestones.
- `glossary.md`: Shared terminology between SSLCommerz, EDD, and this plugin.
- `roles.md`: Behavioral guidelines for AI agents working on this repo.
- `DOCS_POLICY.md`: Rules for keeping this documentation folder synchronized with code changes.
